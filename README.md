## Tech Stack

- Golang
- Air
- HTMX
- TailwindCSS
- Vite


## Run Locally

Clone the project

```bash
  git clone https://github.com/Morpa/htmx-go-todo.git todo
```

Go to the project directory

```bash
  cd todo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  air
```

Start the client

```bash
  npm run dev
```
